# 📱 第五人格トラッカー (Identity V Winrate Tracker)

日々のランク戦の戦績を**「マッチング待機中の数秒」**で記録し、自分の強みや弱点をデータで可視化する、本気で勝率を上げたいプレイヤーのための戦績管理・分析Webアプリです。

🌐 **[アプリを開く（いますぐ使う）](https://idv-tracker.github.io)**

---

##  概要・開発の背景
スプレッドシート等での戦績管理は入力が手間で長続きしない…という課題を解決するため、**「スマホでの圧倒的な入力のしやすさ」**と**「BANピックに直結する高度な分析」**に特化して開発しました。

サバイバー・ハンター両陣営に対応していますが、特に「客観的なデータに基づいてBANや立ち回りを決めたいハンター専プレイヤー」に強くおすすめできる仕様になっています。

##  主な機能

### 1. ランクマに最適な「爆速入力UI」
マッチング中の短い時間で確実に入力できるよう、極限まで無駄を削ぎ落としました。
* **検索付きドロップダウン:** 膨大なキャラクターの中からテキスト検索で一瞬で選択可能。
* **強力な「保持」機能:** 段位、自分のキャラ、特定の味方・相手、マップなど、項目ごとに個別に前回の入力を保持。固定パーティ（デュオ等）での記録も快適です。
* **1タップ記録:** 脱出人数は直感的なボタン（セグメントコントロール）を採用し、プルダウンのストレスを排除。

### 2.  苦手が丸わかり「高度な勝率分析」
蓄積されたデータから、自分の本当の実力とメタ（環境）への適応度を可視化します。
* **詳細な勝率表示:** 「自キャラ別」「マップ別」「対戦相手（編成）別」の勝率・引き分け数を自動計算。
* **ソート機能:** 「勝率順」「試合数順」で昇順/降順の並び替えが可能。「今一番勝てていない苦手キャラ」がワンタップで一番上に表示されます。
* **試合履歴フィルター:** 「特定の相手キャラがいた試合」だけを絞り込み表示でき、具体的な敗因の分析（振り返り）に役立ちます。

### 3.  快適なUXとデータ保護
* **ダークモード対応:** 目に優しいダークテーマを搭載。夜ラン（夜間のランクマッチ）のお供に最適です。
* **データのエクスポート / インポート機能:** 記録した全データはJSON形式でクリップボードやファイルにバックアップ可能。機種変更時や、誤ってブラウザのキャッシュを消してしまった時も安心です。

## 💡 おすすめの運用方法（BAN決めに超役立ちます！）

**「対戦相手別勝率」**タブを使いこなすことで、毎試合のBANピックの精度が劇的に向上します。

> **【活用例】**
> フィルターで **『自分のハンター（例：漁師）』** × **『マップ（例：聖心病院）』** を設定し、勝率を「昇順（↓）」でソート。
> 👉 「漁師で病院の時、相手に祭司がいると極端に勝率が低い」といった**自分だけの苦手データ**が瞬時に可視化されるため、迷わず的確なBANを選択できます。

データが100戦、200戦と貯まるほど、情報の精度が上がり「自分専用の最強のアナリスト」へと進化します！
